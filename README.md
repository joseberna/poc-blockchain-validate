### ⚡ Despliegue Rápido (DevOps)

Este repositorio sigue la convención estándar de Streamlit:
- **Main App**: `streamlit_app.py` (ubicado en el root).
- **Dependencias**: `requirements.txt` (incluye pydantic, rich y streamlit).
- **Estilos**: `config_estilos.css`.

Para desplegar en Streamlit Cloud:
1. Conecta tu cuenta de GitHub.
2. Selecciona este repositorio (`joseberna/poc-blockchain-validate`).
3. El archivo principal ya está configurado como `streamlit_app.py`.
4. ¡Dale a **Deploy**!

### 🚀 Características Maestras v10.6

- **Mempool Dual Timeline**: Visualiza bloques futuros (mempool) y pasados (minados) con diseño 3D box.
